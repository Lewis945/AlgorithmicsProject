Algorithmics project

Project name: Visualization of minimal spanning tree building process and solving TSP.

Team: Olha Shepelenko, Sergeii Shepelenko, Vitalii Zakharov.

Challenge: Visualization of algorithms in user-friendly way in order to be well understood.

Question: How TSP can be solved with a minimum spanning tree?

Description: User can input his data in two ways: file (big data size) or draw a graph in a window. The next step is to build a minimal spanning tree; user can choose one of two algorithms: Prim or Kruskal and watch the building process step by step. When the tree is constructed user can create Hamiltonian cycle by pressing some controls, after that the TSP problem is solved and user can see the shortest path. Probably some optimization will be done if Hamiltonian cycle has crosses. Everything will be done in JavaScript so it is server independent tool that can be opened in any browser without hosting it somewhere.
